# Book_List_Odin

A simple book library web app built with HTML, CSS, and JavaScript.  
Users can add books, mark them as read, and remove them from the library. Humanity reinvented sticky notes, but with gradients and `crypto.randomUUID()`. Progress.

---

## Features

- Add new books using a modal form
- Store:
  - Author name
  - Book title
  - Number of pages
  - Reading status
- Mark unread books as read
- Remove books from the library
- Responsive card-style UI
- Form validation using browser native validation
- Dynamic DOM rendering with JavaScript

---

## Preview

The application displays books as interactive cards with different styles depending on their reading status:

- Green card = already read
- Gray card = unread

Users can:
- Add books
- Change read status
- Remove books instantly without page reload

Because apparently deleting books digitally is less painful than deleting them emotionally.
